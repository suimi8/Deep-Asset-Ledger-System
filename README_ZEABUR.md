# Zeabur 部署指南

本项目支持一键部署至 Zeabur。由于本项目使用 SQLite 数据库，为了确保数据在重启后不丢失，请务必执行以下配置。

## 部署步骤

1.  **上传代码**：将本项目上传至您的 GitHub 仓库并连接至 Zeabur。
2.  **根目录构建**：Zeabur 会自动识别根目录下的 `Dockerfile` 进行多阶段构建。
3.  **配置持久化存储（关键）**：
    *   在 Zeabur 控制面板中，进入您的服务设置。
    *   点击 **"Storage" (存储)** 选项卡。
    *   添加一个新的 **Volume (卷)**。
    *   **Mount Path (挂载路径)** 设置为：`/data`
    *   这将确保数据库文件 `ledger.db` 存储在持久化卷中。
4.  **环境变量**：
    *   默认运行端口为 `8080`。如果需要修改，可以设置环境变量 `PORT`。
    *   数据库路径默认为 `/data/ledger.db`。可以通过 `DATABASE_PATH` 修改。

## 技术架构
- **前端**：React + Vite (静默打包至后端服务)
- **后端**：FastAPI (负责 API 和 静态文件分发)
- **数据库**：SQLite (通过物理卷持久化)
